Under the Hood

Powered by MobileNetV2

LandmarkVision uses a MobileNetV2 convolutional neural network pre-trained on ImageNet and fine-tuned on the Famous Places dataset from Kaggle (~53 MB, 50 world landmarks).

Transfer learning lets us achieve high accuracy with a lightweight model that runs in milliseconds — no GPU required at inference time.

  • MobileNetV2 backbone (ImageNet pre-trained)
  • GlobalAveragePooling + Dense classification head
  • Two-phase fine-tuning (feature extraction → full fine-tune)
  • Data augmentation for robustness
  • 224×224 input resolution
