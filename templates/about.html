The Project

About Landmark Vision

An end-to-end deep learning project — from raw Kaggle images to a live web application.

What is Landmark Vision?

Landmark Vision is a full-stack AI web application that uses computer vision to identify famous world landmarks from uploaded photos. Built with Python, Flask, and TensorFlow, it demonstrates a complete machine learning pipeline — from dataset preparation and model training to web deployment.

The Model — MobileNetV2

The backbone of Landmark Vision is MobileNetV2, a lightweight convolutional neural network architecture designed for mobile and embedded applications. We use transfer learning: MobileNetV2 is pre-trained on ImageNet (1.2M images, 1000 classes) and then fine-tuned on our landmark dataset.

1 Load MobileNetV2 (ImageNet weights, include_top=False)
2 Add GlobalAveragePooling2D → Dense(256) → Dropout(0.5) → Dense(N, softmax)
3 Phase 1: Freeze base — train classification head (10 epochs, lr=1e-3)
4 Phase 2: Unfreeze top 30 layers — fine-tune (10 epochs, lr=1e-5)
5 Save as model/landmark_model.h5

The Dataset

We use the Pictures of Famous Places dataset from Kaggle (vaslemon/pictures-of-famous-places), a curated collection of ~53 MB containing photos of 50 of the world's most famous landmarks. Each landmark has its own folder with multiple images, ideal for fine-tuning.

Property Value
DatasetPictures of Famous Places
SourceKaggle (vaslemon)
Size~53 MB
Classes50 world landmarks
FormatJPG/PNG, folder-per-class
Split80% train / 20% validation
Input Resolution224 × 224 pixels

Download command:

kaggle datasets download -d vaslemon/pictures-of-famous-places -p ./data/
unzip ./data/pictures-of-famous-places.zip -d ./data/landmarks/
